# uditgoenka/autoresearch

**GitHub:** github.com/uditgoenka/autoresearch | **Stars:** ~4K | **Created:** March 2026

## Summary

A Claude Code / OpenCode / Codex "skill" (plugin) that generalizes Karpathy's pattern to any domain. Define a goal with a mechanical metric, then the agent loops: make one change, verify via metric, keep or revert.

## Key Commands

- `/autoresearch` - core modify-verify loop
- `/autoresearch:debug` - scientific-method bug hunting
- `/autoresearch:reason` - adversarial multi-agent debate
- `/autoresearch:ship` - 8-phase shipping workflow

## Relevance to AutoResearch

**Not actually academic research automation** despite the name. It's an iterative code improvement framework. The name overlap is coincidental.

> [!warning] Naming Confusion
> This project shares the "autoresearch" name but does something completely different. It automates code improvement loops, not scientific paper generation.
