# Ivy Collection

**URL:** ivycollection.com

## What It Is

British luxury restaurant group. Referenced as the design inspiration for AutoResearch's color palette.

## Design Extraction

| Token | Hex | Usage in AutoResearch |
|-------|-----|----------------------|
| Forest Green | #245A40 | Primary: buttons, pass states, producer identity |
| Gold/Champagne | #C5A55A | Accent: badges, breakpoints, warnings, gold highlights |
| Cream | #F5F3ED | Page background |
| Warm Taupe | #746144 | Tertiary text, labels |

## Why This Reference

User specifically requested the Ivy Collection palette because it feels "high-end" (gao ji). The green-gold-white combination communicates academic credibility with warmth, avoiding the cold SaaS feel of typical dashboards.

> [!info] Design Decision
> The palette was chosen by the user over the original teal (#0D7C66) + terracotta (#C44D3D) scheme. Typography (Fraunces + DM Sans) was kept from the earlier iteration because Ivy Collection's own fonts have poor readability.
